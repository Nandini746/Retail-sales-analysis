# Customer & Regional Sales EDA

**Project #2: Exploratory Data Analysis (EDA) Dashboard**

---

## 📌 Project Overview

This project analyzes 10,000+ sales transactions to identify:
- Which **regions** generate the highest sales
- Which **customer segments** contribute the most revenue
- How **monthly sales trends** change over time

The goal was to build an interactive dashboard that provides clear, actionable insights for business decision-making.

---

## 🛠️ Tools Used

- **Google Sheets** (PivotTables, Charts, Dashboard)
- **Data Visualization** (Bar, Pie, Line Charts)
- **Microsoft Excel** (Data export)

---

## 📊 Key Insights

| Insight | Finding |
|---------|---------|
| **Top Region** | **[west]** generated the highest sales |
| **Top Customer Segment** | **Consumer** contributed the most revenue |
| **Peak Sales Month** | **[AUgust]** had the highest sales |

---

## 📈 Visualizations

### 1. Sales by Region (Bar Chart)
- **[west]** leads in sales performance
- Provides clear comparison across all 4 regions

### 2. Sales by Customer Segment (Pie Chart)
- **Consumer** segment is the largest revenue driver
- Highlights opportunity to target other segments

### 3. Monthly Sales Trend (Line Chart)
- Sales peak in **[August]**
- Identifies seasonal patterns for planning
