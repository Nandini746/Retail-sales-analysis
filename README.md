# Retail Sales Performance Analysis

## Project Overview
This project analyzes retail sales data to identify which product categories generate the highest revenue. The analysis was done using Microsoft Excel PivotTables.

## Tools Used
- Microsoft Excel (PivotTables)
- Data Visualization

## Key Insights
- Technology category generated the highest sales
- Furniture category was the lowest performer
- Within Furniture, Tables and Bookcases had the lowest sales

## Recommendations
- Focus marketing efforts on Technology products
- Review pricing strategy for Furniture items
- Consider discontinuing or revamping low-performing sub-categories

## How to View
Download the Excel file and open it to see the full analysis with PivotTables.
